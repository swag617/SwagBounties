# Quick Start

Once SwagBounties is installed and running:

## Place a bounty

```
/bounty set <player> <amount>
/bounty set Notch 500
```

Add `--anon` to hide your identity:

```
/bounty set Notch 500 --anon
```

## View bounties

Open the GUI:

```
/bounties
```

Or list bounties on a specific player:

```
/bounty list Notch
```

## Cancel a bounty

```
/bounty remove Notch
```

You will receive a refund minus any configured expiry tax.

## Claim a bounty

Simply kill a player who has an active bounty. The total reward is automatically deposited into your account.
