# Requirements

| Dependency | Version | Required |
|---|---|---|
| Spigot / Paper | 1.21.4 | Yes |
| Java | 21+ | Yes |
| Vault | Latest | Yes |
| Economy plugin | Any Vault-compatible | Yes |
| PlaceholderAPI | 2.11+ | No |

## Notes

- SwagBounties will **refuse to enable** if Vault or an economy provider is not found.
- PlaceholderAPI is soft-depend — the plugin works fine without it, but `%swagbounties_*%` placeholders will not function.
