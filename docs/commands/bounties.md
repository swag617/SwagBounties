# /bounties GUI

```
/bounties
```

Opens the paginated bounty browser GUI.

## GUI Layout

The GUI uses a 54-slot chest inventory:

- **Slots 0–44** — Bounty entries (one skull per target)
- **Slot 45** — Previous page
- **Slot 49** — Page info
- **Slot 53** — Next page

Each skull shows the target's head with:
- Total reward on that player (sum of all bounties)
- Number of active bounties

## Detail View

Click any skull to open a detail view showing each individual bounty on that player:
- Reward amount
- Whether it is anonymous (creator hidden from non-admins)
- When it was placed

Click **Back** to return to the main list.
